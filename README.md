# SolidWorks 工程图一键导出宏使用说明

## 1. 宏功能简介

本宏用于在 SolidWorks 中一键导出当前打开的工程图，并自动分类保存文件。

运行宏后，用户只需要选择或输入一个总文件夹，再输入零件/图纸名称，宏会自动在总文件夹下使用或创建以下 4 个子文件夹：

- `工程图`
- `DWG`
- `PDF`
- `STEP`

导出完成后，文件会分别保存为：

| 文件类型 | 保存位置 | 文件示例 |
|---|---|---|
| SolidWorks 工程图 | `工程图` | `零件名称.slddrw` |
| DWG 图纸 | `DWG` | `零件名称.dwg` |
| PDF 图纸 | `PDF` | `零件名称.pdf` |
| STEP 三维模型 | `STEP` | `零件名称.step` |

## 2. 适用范围

- 适用于 SolidWorks 工程图文件：`.slddrw`
- 已针对 SolidWorks 2019 做兼容处理
- 工程图和 DWG 使用 SolidWorks 另存为方式导出
- PDF 使用 SolidWorks PDF 另存为方式导出
- STEP 从工程图引用的零件或装配体导出

## 3. 使用前准备

使用前请确认：

1. 已安装 SolidWorks。
2. 已打开需要导出的工程图文件。
3. 当前工程图已经保存过。
4. 工程图视图引用的零件或装配体已经保存过。
5. 电脑上已经有本宏的 `.swp` 文件。

建议把宏文件放在固定位置，例如：

```text
D:\SolidWorks宏\工程图一键导出.swp
```

不要放在临时文件夹中，避免后续按钮找不到宏文件。

## 4. 添加到 SolidWorks 工具栏按钮

如果已经有 `.swp` 宏文件，可以按以下步骤添加按钮：

1. 打开 SolidWorks。
2. 点击菜单：`工具 > 自定义`。
3. 打开 `命令` 选项卡。
4. 在左侧选择 `宏`。
5. 将宏按钮拖到工具栏上。
6. 在弹出的设置窗口中选择宏文件：`.swp`。
7. 方法/入口选择：`main`。
8. 如需设置图标，可选择配套的 BMP 图标文件。
9. 点击确定完成。

推荐图标文件：

```text
sw_export_macro_icon_32.bmp
```

如果 SolidWorks 支持更大图标，也可以使用：

```text
sw_export_macro_icon_256.bmp
```

## 5. 正常使用流程

1. 打开需要导出的 SolidWorks 工程图。
2. 点击工具栏上的宏按钮。
3. 宏启动后会先显示版本提示。
4. 选择或输入总文件夹路径。
5. 输入导出的零件/图纸名称。
6. 等待宏自动导出。
7. 导出完成后会弹出完成提示。

当前版本启动提示应显示：

```text
Export bundle v2026-07-14-SW2019-FOLDER-OR-PATH
```

如果没有看到这个版本提示，可能运行的不是当前版本的宏文件。

## 6. 文件夹选择方式

本宏支持两种选择总文件夹的方式。

### 方式一：直接粘贴文件夹路径

运行宏后，第一个输入框可以直接粘贴总文件夹路径，例如：

```text
C:\Users\Public\Desktop\导出文件
```

然后点击确定。

### 方式二：手动选择文件夹

如果想通过窗口选择文件夹：

1. 第一个输入框不要输入任何内容。
2. 直接点击确定。
3. 系统会弹出选择文件夹窗口。
4. 选择总文件夹即可。

注意：选择或输入的是总文件夹，不要直接选择 `工程图`、`DWG`、`PDF`、`STEP` 子文件夹。

## 7. 命名规则

宏会要求输入导出的零件/图纸名称。

例如输入：

```text
料盘
```

则导出结果为：

```text
工程图\料盘.slddrw
DWG\料盘.dwg
PDF\料盘.pdf
STEP\料盘.step
```

如果名称中包含 Windows 不允许的字符，宏会自动替换为下划线 `_`。

不允许的字符包括：

```text
\ / : * ? " < > |
```

## 8. 需要一起分享的文件

建议分享给别人时包含以下文件：

| 文件 | 用途 |
|---|---|
| `.swp` 宏文件 | SolidWorks 可直接运行的宏 |
| `sw_export_macro_icon_32.bmp` | 工具栏按钮图标 |
| `SolidWorks一键导出宏_分享使用说明.md` | 使用说明 |

普通使用者只需要 `.swp` 文件和说明文档即可。

`.bas`、`.swb` 文件主要用于后续修改或重新生成宏，不是普通使用者必须文件。

## 9. 常见问题

### 9.1 提示“没有工程库”

通常是因为按钮绑定的不是 `.swp` 文件，或者 `.swp` 文件损坏。

处理方法：

1. 检查按钮绑定的文件后缀是否为 `.swp`。
2. 不要把按钮绑定到 `.bas` 或 `.swb` 文件。
3. 重新选择正确的 `.swp` 文件。
4. 方法/入口选择 `main`。

### 9.2 点击按钮没有反应

可以检查：

1. SolidWorks 是否启用了宏运行权限。
2. 宏按钮是否绑定到正确的 `.swp` 文件。
3. 方法/入口是否选择了 `main`。
4. `.swp` 文件是否被移动或删除。

### 9.3 提示当前文件不是工程图

请确认当前激活窗口是 SolidWorks 工程图 `.slddrw`，不是零件 `.sldprt` 或装配体 `.sldasm`。

### 9.4 DWG 导出失败

可以先手动执行一次：

```text
文件 > 另存为 > DWG
```

确认 SolidWorks 的 DWG 导出设置窗口已经配置完成，然后再运行宏。

### 9.5 STEP 导出失败

请检查：

1. 工程图视图是否引用了真实零件或装配体。
2. 引用的 `.sldprt` 或 `.sldasm` 是否已经保存。
3. 手动打开该零件或装配体，确认可以另存为 STEP。

### 9.6 PDF 导出失败

请先手动测试当前工程图是否可以：

```text
文件 > 另存为 > PDF
```

如果手动也无法导出 PDF，需要先处理 SolidWorks 本身的 PDF 导出设置或文件问题。

## 10. 日志文件

宏运行时会写入简单日志，方便排查问题。

日志位置：

```text
%TEMP%\SW2019_ExportBundle_Log.txt
```

打开方式：

1. 按 `Win + R`。
2. 输入 `%TEMP%`。
3. 回车。
4. 找到 `SW2019_ExportBundle_Log.txt`。

如果导出失败，可以把日志内容发给维护人员查看。

## 11. 注意事项

- 宏会覆盖同名导出文件。
- 建议工程图、零件、装配体先全部保存后再运行宏。
- 不建议把宏文件放在会经常移动的位置。
- 如果多人使用，建议把 `.swp` 放在统一共享路径或每个人复制到本机固定路径。
- 如果 SolidWorks 版本较旧，首次使用前建议先测试一个简单工程图。

## 12. 推荐分享包结构

建议把文件整理成以下结构后分享：

```text
SolidWorks工程图一键导出宏\
├─ 工程图一键导出.swp
├─ sw_export_macro_icon_32.bmp
├─ sw_export_macro_icon_256.bmp
└─ SolidWorks一键导出宏_分享使用说明.md
```

接收者只需要阅读说明文档，并在 SolidWorks 中把 `.swp` 添加为宏按钮即可使用。
